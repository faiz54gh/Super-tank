#This is where you can add more Tanks. You can freely switch every config position inside Tank setting table
#Beware to not move the bracket '{}'. You can add your own comments by using # then follow by your comment texts.
#DO NOT add comment the same line of tank setting. Below is what does each setting do.
#-------------------------------------------------------------------------
#Base_Damage = Tank Damage in  in %. eg Base_Damage = 40 means Tank will do 40% of original damage.
#Base_Health = Tank Health. Leave to 4000 to use original game health.
#Health_Multiply = Multiplies Tank Health with this value. eg. set to 1.0 means Tank will have 100% more health.
#Enable = Enable or Disable this tank from spawning. Set to 'true' and 'false' only
#Fire_Immune = Enable or Disable fire immunity on this entity. Set to 'true' and 'false' only.
#Variant_Fire_Immune = Enable or Disable fire immunity on this Tank variant. Set to 'true' and 'false' only.
#Overlord_Chance = Chance to spawn overlord class of this Tank.
#Difficulty_Appearance = Allow this Tank to be spawned on specific difficulty.
#	5 = All Difficulty
#	4 = Expert
#	3 = Advanced
#	2 = Normal
#	1 = Easy
#Can put | sign to allow more than one difficulty eg 1|2|3 This will allow Tank to be spawned on Easy, Normal and Advanced.
#Finale_Only: Allow this Tank to be spawned in finale chapter only. Set to 'true' or 'false' only.
#-------------------------------------------------------------------------
